# UnrealAny
基于UE5的自定义结构体FAny

# 支持类型
* bool
* int
* int64
* float
* double
* FString
* FName
* FText
* FVector
* FRotator
* FTransform
* 自定义反射结构体
* 自定义反射枚举
* Uclass
* 基于UObject的对象指针
